Spend Waterfall Analysis

The Spend Waterfall Report is only available in advertiser and agency nodes.

The Spend Waterfall Report provides a near-real-time view of the costs of each component of your supply chain for a specified timeframe. Some of the cost data is collected by log files and some of the cost data will need to be provided by estimates in order to get a complete picture–because some cost data is not available from log files.

Costs

The following section provides a decription for each of the costs provided in this report.

Name Description
Agency The cost from the agencies services. This data is not available from log files.
Ad Server The costs incurred from the ad server (such as Google CM 360 or Flashtalking). This data is not available from log files.
CV The costs incurred from the content verification (CV) tool (such as DoubleVerify, IAS or MOAT) . This data is not available from log files.
DSP Platform The fee that the DSP charges for using its platform
DSP Data The cost that the DSP charges for using data on its platform.
DSP Other This includes any fees that the DSP may charge outside of platform, data, or media.
DSP Media Cost This is the cost that the DSP paid to purchase the impression from the SSP, as reported by the DSP.
Media Cost Delta This is the difference between the DSP media cost and the SSP cost (if there is one)
SSP This is the total cost that the SSP charged the DSP.
Gross Working Spend The total amount that was paid to the publisher.
Unmatched with CV The amount that was paid to the publisher that was not matched with the CV log files.
Fraud The amount that was paid to the publisher and was determined by the CV tool to be fraudulent.
Non-brand Safe The amount that was paid to the publisher and was determined by the CV tool to be non-brand safe.
Non-Viewable The amount that was paid to the publisher and was determined by the CV tool to be non-viewable.
Viewable The amount that was paid to the publisher and was determined by the CV tool to be non-fruadulent, brand safe, and viewable.

The following table shows which options are available for estimating costs in this analysis

Cost Use data from log files collected by TAG TrustNet % of spend estimate CPM Estimate

SSP X X X

Use Data from Log Files Collected by TAG TrustNet

This option will automatically base the cost on the log file data that is collected by TAG TrustNet from the vendors (including the DSP, SSP, or Publisher)

% of Spend Estimate

You can enter a % of spend estimate for this cost based on historical or expected costs.

CPM Estimate

You can enter a CPM estimate for this cost based on historical or expected costs.

Controls

Limit to SSP data available

Using this option will allow you to only see the spend data where there is matching SSP data. This option is available to provide you more relevant data.

Settings

Clicking this button will allow you to set the source of data that will be used for this analysis, and well as input specific values to be used for the estimate (if available).
