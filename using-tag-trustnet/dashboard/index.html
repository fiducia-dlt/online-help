Dashboard

The Dashboard screen is only available on advertiser or agency nodes.

The purpose of the dashboard is to allow you to easily see the most critical metrics that are tracked by TAG TrustNet everytime you login. For most metrics you have the option to set a goal so that you can see how well these metrics are doing against your goals.

The dashboard is comprised of a number of individual panels showing metrics and goals called widgets. These are grouped into two categories, Ad Effectiveness and Cost Efficiency. The widgets can be added, changed, and deleted, in order to customize the dashboard for your account. The following provides information about how to make these changes.

Time Period

The time period for data that is used for the entire dashboard is set at the upper left of the dashboard. You can set this to any time period that you want and this setting will be used when you login in the future.

Change Existing Widgets

Modify Existing Widgets

To change the settings for an existing widget:

  1. Within the widget that you want to change, click on the icon in the top-right corner with the three vertical dots.
  2. Select Edit
  3. Make the desired changes
  4. Click the Save Widget button.
Move Existing Widgets

You can easily change the position of widgets on the dashboard by doing the following:

  1. Hover over the widget that you want to move.
  2. A small grid of six dots will appear to the left of the widget name.
  3. Click (and hold) on top of the dots and move the widget to the desired location.
  4. Once the widget is at the desire location release the mouse button.

Note: You are not allowed to move widgets to different categories (for example from Ad Effectiveness to Cost Efficiency, or vice-versa).

Delete Existing Widgets

In order to delete a widget follow these steps:

  1. Within the widget that you want to change, click on the icon in the top-right corner with the three vertical dots.
  2. Select Remove.

Add New Widgets

To add a widget follow these steps:

  1. At the top-right of the dashboard screen click on the + Add Widget button.
  2. Select whether you want it to be a a Goal Widget or Over Time widget. As you select these options you can see a preview of what they will look like to the right.
  3. Select the metric that you want to use from the drop-down menu.
  4. Select whether you want to use Ad Spend or Impressions as the basis for the metrics.
  5. Select whether you want to show the metric value or a % in the widget (the preview will show how these options will be displayed)
  6. Choose a goal to be used to measure the performance of the metric. This is optional.
  7. Click the Add Widget button.

Once the widget is created you can move it to the desired location in the dashboard. Please refer to Move Existing Widgets above.

Note: most, but not all settings can be changed once you have created a widget. Once you have selected a widget to be a Goal Widget or Over Time widget you cannot change it to be the other type once you select Add Widget. If you want to make this change your only option is to delete the widget and create another one.
