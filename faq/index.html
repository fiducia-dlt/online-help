Release Notes

(Minor) 1.4 Released October 17, 2023

New (Advertiser and Agency Nodes)

● Support for TAG Certified For Transparency has been added to the Supply Chain Report and to the CSV file for the Supply Report.

(Major) 1.3 Released May 5, 2023

New (Advertiser and Agency Nodes)

● For the dashboard and all insights, you can now select multiple (or all) advertiser IDs and campaign IDs.

● In the Supply Report there are some new filters, Ad Verification Match and Exchange Match. These will allow you to view only impressions where there was a match between the DSP and Ad Verification platforms, or DSP and Exchange platform respectively.

● In the Supply report the following columns have been added to the screen and CSV file: DSP Platform Fee, DSP Data Fee, DSP Other Fee, and DSP Media Cost.

● TrueCPM, TrueAdSpend, and TrueImpressions have replaced Qualified CPM, Qualified Ad Spend, and Qualified Impressions. Note: this change may be delayed for some customers.

(Minor) 1.2 Released April 18, 2023

New (Advertiser and Agency Nodes)

● In the Dashboard, you are now able to create a widget to monitor your Qualified CPM. To do this follow these steps. When creating a new widget, select Qualified as the Metric. In the Value menu select CPM.

(Major) 1.1 Released January 24, 2023

New (Advertiser and Agency Nodes)

● In the Supply Chain Report, when you are viewing the Qualified metric you will also see Qualified CPM data in the tool tips box when hovering over the supply chain paths between two vendors.

● In the Supply Report the following columns have been added: CPM, Qualified CPM, and Qualified Viewable CPM

● In the Data Feeds menu there is a new option, titled the Creative Match Report. The purpose of this report is to allow you to see which creatives are causing match issues between your DSP and Ad Verification data. The creatives with a 0% match rate should be investigated to see if the tag to pass the impression ID to ther Ad Verification system was properly implemented. This report, which includes the seat, campaign, and creative IDs, can be downloaded and sent to the people managing the creatives.
